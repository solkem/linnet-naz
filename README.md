# NAZ ICTD Interview Prep App

One-page interactive web app to prepare for the Nutrition Action Zimbabwe ICT4D technical assessment.

## Local run

Open `index.html` in a browser.

## GitHub Pages

This repository auto-deploys to GitHub Pages via `.github/workflows/deploy-pages.yml` on every push to `main`.

Expected URL:

- `https://solkem.github.io/linnet-naz/`
